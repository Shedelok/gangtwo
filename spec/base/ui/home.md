# About This File

This file describes the UI of the home page.

# Name Text Input

When a player joins the lobby by submitting their name and pressing the Join Lobby button, their name is remembered by
the application at the browser level. Every time that player opens the home page of the game, their last used name
is prefilled into the text input, so that they can join the lobby with the same name they used last time with just one
click. The name should be prefilled in all cases. The name should be prefilled when Stop the Game button is pressed
during a game. The name should be prefilled when Stop the Game button is pressed during the lobby phase. The name should
be prefilled when the player refreshes the browser page during the lobby phase.

# Join Lobby Button

Join lobby button has text "Join Lobby (X player(s))", where X is the number of players currently in the lobby and
"player(s)" is actually either "player" or "players" depending on the number.
